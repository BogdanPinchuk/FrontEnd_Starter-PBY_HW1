Теги физического формирования:

< b >   придает тексту жирное начертание
< i >   придает тексту курсивное начертание
< u >   делает текст подчеркнутым
< s >   отображает текст как перечеркнутый
< bdi >   указывает фрагмент текста, который должен быть изолирован от изменения направления вывода текста
< bdo dir="ltr" >   устанавливает направление вывода текста
< bdo dir="rtl" >   устанавливает направление вывода текста
< small >   делает текст на одну условную единицу меньше
< big >   делает текст на одну условную единицу больше
< sup >   помещает содержимое в верхний индекс
< sub >   помещает содержимое в верхний индекс
< meter >   используется для вывода значения в некотором известном диапазоне
< progress >   используется для отображения прогресса завершенности
